Technologies

Mostly versed in front-end development, I currently have a good feeling with Ionic 2 and Angular 2, but like also React. I used to work a lot with Angular 1, but if a project can afford more recent framework, then go for it.

I love playing with HTML, CSS, and JavaScript for user interface and interactions. I care a lot about accessibility and semantic (I even think you can have a dedicated team member just about those parts). I value a lot user experience and am absolutely delighted to work with good UX designer! I must admit, my design and UX background might have a bit tweaked my opinion on the subject...

I've also spent time around front-end web-mapping and Leaflet is the framework to go with at the moment. However, keep an eye on MapboxGl !

About backend technologies, I must say Python and Django have my attention at the moment.

Git for versioning, automatised tests for quality and agile for project management

Currently an Archlinux fanboy for coding, even if I'm still using OSX for music and Windows for games....
